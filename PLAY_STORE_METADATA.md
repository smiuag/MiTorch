## Long Description
**Max: 4000 characters** (full app description on Play Store)

```
BlowTorch is a feature-rich Telnet/MUD client for Android, designed for both sighted and blind users.

FEATURES:
• Terminal emulation with ANSI color support
• Customizable button grids for frequently used commands
• Interactive visual map system with room navigation
• Channel system for organizing game communication
• Blind Mode with TalkBack accessibility and voice feedback
• GMCP protocol support for structured game data
• Command history and auto-completion
• Multiple server configuration and quick-switch
• Auto-login with saved credentials
• Sound notifications for important events
• Compact and performance-optimized interface

BLIND MODE HIGHLIGHTS:
• Full TalkBack screen reader support
• Voice announcements for game status
• Silent mode toggle for uninterrupted gameplay
• Accessible button layouts with secondary actions
• Filter system for auto-silencing game prompts
• Multi-panel interface for organized information

REQUIREMENTS:
• Android 6.0 (API 24) or higher
• Internet connection (WiFi or mobile data)
• Telnet-based MUD server to connect to

PERMISSIONS:
• Internet: Required for Telnet connections
• Notifications: For connection and game alerts
• Wake Lock: To maintain connection while device sleeps

NOTE:
This is a Telnet client for MUD (Multi-User Dungeon) games and text-based services. Each MUD server operates independently with its own rules, content, and terms of service. Please review the specific server's terms before connecting.

BlowTorch is open source and constantly improving. Feedback and suggestions are welcome.
```
